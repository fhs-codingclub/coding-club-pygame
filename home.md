# Game Overview *GAME NAME*
**Genre** Top-down Python RPG
**Current Version** Pre-alpha 

## Current State
Currently in Pre-Alpha, the goal of this is to create a basic playable game. This includes a basic combat system, simple exploration, placeholder art, and basic enemy encounters. 

## Core Idea
Players will explore an massive underground/mountanous civilization connected to an academy focused on dragons. The deeper they get into the area the more difficult the enemies get, and the more lore they uncover about the dragons and this old academy. They learn about the dragonborn people and learn about the truth behind an ancient evil released a long time ago. 

The game will combine turn based combat, exploration (like metrovania's (IE, you have to get an upgrade to progress)), and character interaction. Progressing through the new areas gives them new abilities, items, and experience to upgrade their characters. Thus they will be able to battle stronger enemies and uncover new ways to get deeper in the area. 

Different playable characters with different playstyles will be included to offer different playstyles, encouraging multiple playthroughs. (If we are able to multiple endings)

## Gameplay Loop
Explore New areas. Interact with npc's/items. Enter combat and skill challenges. Earn items, upgrades, and experience. Unlock abilities and equipment. Access areas and lore. Repeat

## Tone/Setting 
The game takes place in a mountanous/underground region that used to be this dragon academy. It seems that the academy possesed some kind of "Dark Academia" vibe but after the many years, it has been deserted and is older now. Ancient texts, statues, laboratories, settlements, some possessed, hint at a long troubled history with the dragons and humans. 

The more that players explore, the more they learn about the lore of the world they live in.
